El Poder del Determinante (Regla de Cramer)

Resolver un sistema 3x3 significa encontrar el punto exacto en el espacio tridimensional donde tres planos se intersectan. La Regla de Cramer es el método más estructurado y elegante para lograrlo.

\[ x = \frac{\Delta x}{\Delta s}, \quad y = \frac{\Delta y}{\Delta s}, \quad z = \frac{\Delta z}{\Delta s} \]

Donde \(\Delta s\) es el determinante del sistema. Si \(\Delta s \neq 0\), el sistema tiene una solución única.

Ejemplos Guiados

Ejemplo 1: Cálculo del Determinante del Sistema (\(\Delta s\))

Dado el sistema:

\( \begin{cases} x + y + z = 6 \\ 2x - y + z = 3 \\ x + 2y - z = 2 \end{cases} \)
Paso 1: Extraer los coeficientes de las variables (x, y, z).
Paso 2: Formar la matriz y resolver por regla de Sarrus o cofactores.
\[ \Delta s = 1(1-2) - 1(-2-1) + 1(4 - (-1)) = -1 + 3 + 5 = 7 \]

Ejemplo 2: Encontrar una Variable (\(x\))

Usando el mismo sistema, calculemos \(\Delta x\). Sustituimos la columna de las \(x\) por los resultados (6, 3, 2).

Paso 1: Formar el determinante de x (\(\Delta x\)).
\[ \Delta x = 6(1-2) - 1(-3-2) + 1(6 - (-2)) = -6 + 5 + 8 = 7 \]
Paso 2: Dividir \(\Delta x\) entre \(\Delta s\).
\[ x = \frac{7}{7} = 1 \]

Ejemplo 3: Método de Reducción (Gauss)

Alternativamente, podemos sumar o restar ecuaciones para eliminar variables. Si sumamos la Ecuación 1 y la Ecuación 2 del ejemplo anterior:

Eq 1: \(x + y + z = 6\)
Eq 2: \(2x - y + z = 3\)
Suma: \(3x + 2z = 9\)
Nota: Hemos eliminado la variable \(y\). Repitiendo este proceso con otras ecuaciones, reducimos el sistema a 2x2.

Desafíos de Nivel Genio

Puntos de Genio: 0 / 3

Desafío 1: El Determinante

Calcula el determinante del sistema (\(\Delta s\)) para la siguiente matriz de coeficientes:

\( \begin{vmatrix} 2 & 1 & -1 \\ 1 & 2 & 1 \\ 3 & -1 & 2 \end{vmatrix} \)

Desafío 2: Despeje Final

En un sistema, calculaste que el determinante del sistema es \(\Delta s = 10\) y el determinante de \(y\) es \(\Delta y = 50\). ¿Cuál es el valor de \(y\)?

Desafío 3: Resolución por Reducción

Observa estas dos ecuaciones simplificadas de un sistema:
1) \(x + z = 4\)
2) \(3x = 3\)
¿Cuál es el valor exacto de la incógnita \(z\)?
