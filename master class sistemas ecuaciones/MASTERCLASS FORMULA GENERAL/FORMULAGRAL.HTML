1. Introducción al Tema

Una ecuación cuadrática tiene la forma estándar: $$ax^2 + bx + c = 0$$

por ejemplo:$$5x^2 - 8x + 40 = 0$$

  1. $a=5$
  2. $b=-8$
  3. $c=40$

La Fórmula General es la herramienta definitiva (nuestra "llave maestra") para encontrar los valores de $x$ (las raíces) sin importar si la ecuación es factorizable o no. Su estructura es:

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

El término $b^2 - 4ac$ se llama discriminante. Si es positivo, hay 2 soluciones; si es cero, 1 solución; si es negativo, no hay soluciones reales.

2. Modelaje: Ejemplos Paso a Paso

Ejemplo 1

$$x^2 - 5x + 6 = 0$$

a=1, b=-5, c=6
$$x = \frac{-(-5) \pm \sqrt{(-5)^2 - 4(1)(6)}}{2(1)}$$
$$x = \frac{5 \pm \sqrt{25 - 24}}{2} = \frac{5 \pm 1}{2}$$
$x_1 = 3$, $x_2 = 2$

Ejemplo 2

$$2x^2 + 4x - 6 = 0$$

a=2, b=4, c=-6
$$x = \frac{-4 \pm \sqrt{(4)^2 - 4(2)(-6)}}{2(2)}$$
$$x = \frac{-4 \pm \sqrt{16 + 48}}{4} = \frac{-4 \pm 8}{4}$$
$x_1 = 1$, $x_2 = -3$

Ejemplo 3 (Raíz única)

$$x^2 - 4x + 4 = 0$$

a=1, b=-4, c=4
$$x = \frac{-(-4) \pm \sqrt{(-4)^2 - 4(1)(4)}}{2(1)}$$
$$x = \frac{4 \pm \sqrt{16 - 16}}{2} = \frac{4 \pm 0}{2}$$
$x_1 = x_2 = 2$

3. Práctica Guiada: Ejercicios Resueltos

Ejercicio 1

$$x^2 + 7x + 10 = 0$$

a=1, b=7, c=10

$$x = \frac{-7 \pm \sqrt{49 - 40}}{2}$$

$$x = \frac{-7 \pm 3}{2}$$

$x_1 = -2, x_2 = -5$

Ejercicio 2

$$3x^2 - 12x + 9 = 0$$

a=3, b=-12, c=9

$$x = \frac{12 \pm \sqrt{144 - 108}}{6}$$

$$x = \frac{12 \pm 6}{6}$$

$x_1 = 3, x_2 = 1$

Ejercicio 3

$$x^2 - 8x + 15 = 0$$

a=1, b=-8, c=15

$$x = \frac{8 \pm \sqrt{64 - 60}}{2}$$

$$x = \frac{8 \pm 2}{2}$$

$x_1 = 5, x_2 = 3$
