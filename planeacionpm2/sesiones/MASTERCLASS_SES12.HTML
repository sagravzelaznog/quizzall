Historia y Evolución Numérica

Del conteo con piedras a la abstracción digital

01. ¿Qué es un Sistema Numérico?

Un sistema de numeración es un conjunto de símbolos y reglas que permiten representar datos numéricos. A lo largo de los milenios, la humanidad ha transitado desde sistemas aditivos (como el egipcio) hasta sistemas posicionales (como el nuestro), donde el valor de un símbolo depende de su ubicación.

02. Secuencia de Aprendizaje

Inicio: El Trueque Prehistórico

Simulación de intercambio de bienes sin usar nombres de números, solo correspondencia uno a uno (piedras vs ovejas).

Desarrollo: La Revolución del Cero

Análisis de cómo las civilizaciones Maya e Hindú transformaron la aritmética al definir "la nada" como un valor.

Cierre: El Código del Futuro

Exploración breve del sistema binario ($0, 1$) como la base de toda la tecnología actual.

1. Sistema Egipcio: Es puramente aditivo. Para escribir 23, se repetía dos veces el símbolo de diez y tres veces el de uno.

2. Sistema Romano: Mixto (aditivo y sustractivo). Ejemplo: $IV = 5 - 1 = 4$.

3. Sistema Maya: Vigesimal (base 20) y posicional de forma vertical.

  1. Expresa el número actual de tu año de nacimiento usando numeración Romana.
  2. Si el sistema binario usa base 2, ¿cuál es el valor decimal del número $101_2$?
  3. Diseña un símbolo para representar el número "12" en un sistema de base 12.
